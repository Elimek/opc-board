🚀 Quick Start

Deploy Your Board in 2 Minutes

# Clone your board of directors
$ git clone https://github.com/Elimek/opc-board.git
$ cd opc-board

# Option A: Deploy to GitHub Pages (free)
$ git push origin master
# → Settings → Pages → select root → wait 2 min

# Option B: Open locally
$ open index.html

# Option C: Deploy to Vercel (free)
$ npx vercel --prod
