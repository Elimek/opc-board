## 5个核心心智模型

### 1. 功率定律（Power Laws）
不是所有健康行为都平等。排序是：
1. 不吸烟 >> 2. 不酗酒 >> 3. 睡8小时 >> 4. 全食物饮食 >> 5. 每天运动
先搞定前三项，再考虑补充剂。90%的人都卡在第三步之前。

### 2. 测量即管理（Don't Guess, Measure）
- 睡眠：Oura Ring 或 Apple Watch（每日评分）
- 身体：DEXA 扫描（每季度体脂/肌肉/骨密度）
- 血液：全血检（每季度 CBC、代谢、炎症、激素）
- 表现：VO2 max（每年）、握力（每月）
- 生物年龄：每年用多个时钟测量
- 最低起步：睡眠评分 + HRV + 静息心率

### 3. 协议栈化
不要每天靠意志力做决定。把健康行为协议化为固定程序：
- 早晨协议栈 → 光照、水温、补充剂、运动顺序固定
- 饮食协议栈 → 固定菜单，不每天纠结
- 睡眠协议栈 → 固定时间、固定程序、固定环境
协议栈的目标是：**把正确的事情变成自动化行为。**

### 4. 复利健康
今天的每一个健康决策，都乘以未来的时间。一天高质量的睡眠不改变什么，但每晚7小时vs8小时，10年后相差7300小时——这相当于改变了你的生物学年龄。健康复利是最被低估的复利。

### 5. 算法优于意志力
意志力是有限的消耗品。不要依靠自律，要依靠系统：
- 环境设计：冰箱不放垃圾食品
- 时间固定：每天同一时间睡觉
- 追踪可视化：看到数据比下决心更有效
- 社会承诺：公开承诺比自我约定有效
