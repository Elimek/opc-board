# 🌅 Daily OS — Operating System

> The body is the company's infrastructure. Protect it.

## Morning (6:00-8:00) — Foundation

| Time | Action | Check |
|------|--------|-------|
| Wake | No phone. No caffeine yet. | ☐ |
| +0min | 5-10 min sunlight (nature or bright light device) | ☐ |
| +10min | Water with electrolytes | ☐ |
| +15min | Cold shower (1-3 min) or face cold plunge | ☐ |
| +90min | Caffeine (coffee/tea) | ☐ |
| | **Deep Work Block #1** (60-90 min, phone away) | ☐ |

## Midday (12:00-14:00) — Refuel

| Time | Action | Check |
|------|--------|-------|
| 12:00 | Lunch (protein + veggies, no processed food) | ☐ |
| 12:30 | Walk (10-20 min) or NSDR (10-20 min) | ☐ |
| 13:00 | Shallow work: email, social, coordination | ☐ |

## Afternoon (14:00-18:00) — Produce

| Time | Action | Check |
|------|--------|-------|
| 14:00 | **Deep Work Block #2** (60-90 min) | ☐ |
| | Exercise: strength (3x/wk), cardio (2x/wk), mobility (daily) | ☐ |
| 15:00 | No caffeine after 3pm | ☐ |
| | Learning / reading (30 min) | ☐ |

## Evening (18:00-22:00) — Wind Down

| Time | Action | Check |
|------|--------|-------|
| 18:00 | Dinner (light, finish 3h before sleep) | ☐ |
| 20:00 | Dim lights, no screens if possible | ☐ |
| 21:00 | Journal / review day | ☐ |
| 21:30 | Plan tomorrow's 3 MITs | ☐ |
| 22:00 | Sleep (aim for 8+ hours) | ☐ |
